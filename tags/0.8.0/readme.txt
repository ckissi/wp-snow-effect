== Description == 

Add nice looking animation effect of falling snow to your Wordpress site and enjoy winter. This WP plugin uses jSnow JQuery plugin (2kb only) and no images.

== Installation == 

1. Drop the 'wp-snow-effect' folder into your 'wp-content/plugins' folder
 
2. Go to your Administration: Plugins page and activate the "WP snow effect" plugin
